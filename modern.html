Spawns & Signals

Right-click a track tile to spawn a train, add a block signal, or add a distant signal — each facing either way. A signal protects the block ahead (connected track up to the next signal or dead end); mains are green when the block is clear, red when taken. Place a distant one or more tiles before a signal so a train can roll through on green; a signal with no distant shows a hollow caution marker and trains always brake to it and pause briefly before going. Caution tiles (orange outline) make trains crawl and stop safely; on normal track a meeting is a crash.

Train types

A layout has a list of train types, each a numeric id with a display colour and optional name. Spawns and filters refer to a type by id, so recolouring a type never changes routing. Click a swatch to use that type for new spawns; edit the colour or name inline; use + to add a type and × to remove an unused one.
