Spawns & Signals

Right-click a track tile to spawn a train or add a block signal facing either way. Automatic signals protect a block: green when clear, red when taken — a train rolls through a green one without slowing and brakes to a stop at a red one. An automatic main must have an opposing main on the same tile (use + both); without one it blinks red and no train may pass (it would otherwise lead a train out of a protected block into open track). Manual signals are default red and have no block; in Operate mode click one to clear a route — allowed only when the path through to a signal facing your way is clear and crosses no already-locked switch. Clearing locks the switches on the route; the signal drops to red as a train passes and each switch unlocks as the train's tail clears it. All trains run on sight and never crash: a train only stops for an occupied tile ahead, a manual switch set against it, or a red signal.

Train types

A layout has a list of train types, each a numeric id with a display colour and optional name (purely cosmetic now — routing is by switch setting, not type). Click a swatch to use that type for new spawns; edit the colour or name inline; use + to add a type and × to remove an unused one.

Stations

Pick the Station tool and drag a rectangle to mark a named region. Name the stops, switches and signals inside it (right-click an element → Name) using station-local labels like 1E, 2W, A, 2 — these are how a station master refers to the infrastructure. Rename or remove stations below.
